# blotter

A browser-only blotter art filter generator with a retro folder-style interface.

## Use

Open `index.html` in a modern browser, choose an image, adjust the four controls, then save the generated PNG.

## Publish with GitHub Pages

1. Create a new GitHub repository and upload this folder's contents.
2. In the repository, open **Settings → Pages**.
3. Deploy from the `main` branch and the repository root (`/`).

No build tools, dependencies, or server are required.
