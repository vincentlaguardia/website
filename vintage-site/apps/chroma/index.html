this app is a color study

nine circles share one color. same hex, same value. but each one appears to change because it sits inside a different palette.

that trick is called simultaneous contrast. a color is never only itself, it is shaped by the colors around it.

every hue lives somewhere on a circle from 0 to 360 degrees. red is near 0, yellow near 60, green near 120, cyan near 180, blue near 240, magenta near 300, then back to red again.

a palette philosophy is a rule for choosing colors from that circle.

monochromatic stays with one hue, changing only lightness and saturation.

analogous uses nearby hues, so the palette feels close and natural.

complementary pairs opposite hues for clear contrast.

split complementary uses two colors near the opposite hue, keeping contrast while feeling a little softer.

triadic spaces three hues evenly around the wheel.

tetradic uses four positions, giving the palette more energy and variety.

hue chooses the starting point. strange decides how closely the grid follows the rule.

low strange keeps things orderly. high strange lets colors wander.

the grid is arranged in diagonal bands, so colors gather into small neighborhoods instead of scattering at random. choose a style, reroll the hue, sample a color, copy its hex or rgb, and see how the same color changes from place to place.

file>import photo distills an image into Chroma's color energy. export unique palette hex values from file>export palette or use the Export Palette button.

dr pastrami mode is something else entirely. it does not follow the normal rules of saturation or lightness. it knows where pastels and neons should never meet, and puts them there anyway. not every hue combination it finds has a name. use it carefully.
