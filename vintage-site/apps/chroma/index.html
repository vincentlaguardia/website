Theory
X

techno mode is a parameterized color system built around controlled conflict. it begins with a conventional palette structure, distributes color families through a fixed modular diagonal field, and then assigns colors to two contrasting states: pastel and neon. these states cross the original hue relationships, allowing the system to create tension without losing its underlying organization.

its defining mechanism is bounded deviation. colors remain connected to an assigned source family, but they are permitted to move away from it within controlled limits. the field, state system, distribution weights, and repeated contextual circles continue to organize the composition even as individual colors become less conventionally harmonious. this allows techno mode to preserve a recognizable structural identity while producing unstable, synthetic, and visually charged results.

techno mode combines several established color principles, including contrast, proportion, simultaneous contrast, hue relationships, and contextual perception, inside one procedural system. what distinguishes it is the way these elements operate together. the modular field determines spatial ancestry, pastel and neon states control chromatic behavior, strange controls permitted departure, skew controls distribution, and constant circles reveal how the surrounding field changes perception. the result is a color system in which harmony functions as a starting grammar rather than a final constraint.

Techno

abstract

techno mode treats color not as a settled agreement but as a field where order and rupture coexist. within a fixed 7 × 7 grid, harmony is not the destination; it is the remembered origin from which the image is allowed to drift. four mechanisms govern that drift: a restricted set of source harmony grammars; a bimodal pastel and neon state model; a normalized strange parameter controlling chromatic deviation; and a deterministic modular diagonal field that assigns cells to four repeating bands. the result is controlled visual conflict: color permitted to wander, but never permitted to forget the structure that contains it.

where conventional harmony systems ask how colors resolve, techno mode asks how they remain intelligible at the edge of resolution. its central claim is that structure and instability are not opposites but collaborators: the grid preserves identity precisely so the palette can risk mutation. this paper specifies the model, distinguishes source harmony from generated palette geometry, introduces measures for grammar deviation and local perceptual conflict, and defines the conditions under which this instability remains reproducible, bounded, and open to empirical study.

1. introduction

computational color systems must negotiate a recurring tension between coherence and variation. highly constrained systems, such as fixed complementary or triadic schemes, produce recognizable relationships but may limit expressive range. unbounded random sampling produces greater variation but can weaken compositional identity.

techno mode occupies the interval between those conditions. it preserves an invariant spatial substrate while expanding the admissible behavior of color within it. grid geometry and band membership remain deterministic. harmony family, hue mutation, chromatic state, and per cell variation determine the generated surface.

the resulting model can be summarized as:

controlled visual conflict = structural invariance + bounded chromatic deviation

here, conflict does not mean arbitrary discord. it refers to perceptual differences that depart from an initial palette grammar while remaining statistically bounded and spatially organized.

techno mode is best understood as a parametric color composition system, rather than solely as a traditional color harmony theory. traditional theories generally ask which colors belong together. techno mode asks a different question:

how far can a palette depart from an underlying color relationship while the composition retains an identifiable structural character?

2. conceptual position

techno mode draws upon several established ideas without being reducible to any one of them:

  • traditional harmony families supply source relationships among hues.
  • perceptual color spaces make chromatic differences measurable.
  • modular arithmetic supplies a deterministic spatial field.
  • mixture distributions produce distinct pastel and neon populations.
  • constrained stochastic generation produces variation without altering the underlying topology.

the distinctive contribution is the architecture connecting these components. harmony is treated as a source grammar, not a guaranteed description of the final palette. randomness operates only in the color layer and never modifies spatial band membership. a single control variable increases the permitted deviation while preserving the compositional scaffold.

this position differs from color harmonization methods that optimize an existing image toward a recognized harmony template. techno mode permits movement away from such a template and treats the resulting tension as an intended feature.

3. structural substrate

3.1 grid

the reference composition uses a 7 × 7 grid containing 49 cells. nine cells contain target dots and 40 act as background or support cells:

9/49 ≈ 18.4%    40/49 ≈ 81.6%

because support cells occupy most of the field, the overall composition is driven primarily by relationships among background colors, not by the target dots alone.

the 7 × 7 configuration is the reference implementation rather than a mathematical necessity. a generalized implementation may use an r × c grid, provided the spatial and chromatic stages remain separate.

3.2 modular diagonal band field

for zero indexed row r, column c, and band count n, band membership is:

b(r,c) = (r+c) mod n

for the reference configuration, n = 4. the grid therefore begins:

0 1 2 3 0 1 2
1 2 3 0 1 2 3
2 3 0 1 2 3 0
3 0 1 2 3 0 1
0 1 2 3 0 1 2
1 2 3 0 1 2 3
2 3 0 1 2 3 0

cells with equal values of (r+c) mod n occupy a repeating diagonal residue class. this construction is described here as a modular diagonal band field. when the grid and band count are both n, the same formula generates a cyclic latin square. the 7 × 7, four band reference case is periodic but is not itself a latin square because symbols repeat within individual rows and columns.

diagonal repetition is used because it produces a woven directional rhythm while remaining deterministic. horizontal and vertical neighbors always advance by one band modulo n. consequently, the field encodes a predictable adjacency structure before any colors are generated.

4. source harmony grammar

the full application recognizes six conventional palette families:

  1. monochromatic;
  2. analogous;
  3. complementary;
  4. split complementary;
  5. triadic; and
  6. tetradic.

techno mode admits four: monochromatic, analogous, split complementary, and tetradic. complementary and triadic are excluded.

admitted fraction = 4/6 ≈ 66.7%    excluded fraction = 2/6 ≈ 33.3%

this restriction should not be interpreted as proof that complementary and triadic palettes are universally more resolved than other families. rather, it is an explicit design decision: in the implementation, these two families provide strongly legible symmetric intervals that work against the desired aesthetic of asymmetrical mutation.

the chosen family defines the source grammar g, consisting of a set of nominal hue positions: g = {g₁, g₂, …, gₖ} where gᵢ ∈ [0, 360).

after mutation, the generated hues need not retain the conventional label of the source grammar. the system therefore distinguishes between:

  • source grammar: the relationship used to initialize the palette;
  • generated geometry: the relationships present after mutation; and
  • grammar deviation: the measured distance between the two.

this distinction prevents a heavily mutated palette from being described as analogous or tetradic when its final geometry no longer supports that description.

5. bimodal chromatic states

each generated cell occupies one of two states. in the reference hsl implementation, their ranges are:

statesaturationlightnessapproximate mean
pastel50 to 8072 to 88s=65, l=80
neon92 to 10047 to 67s=96, l=57

the mean separation is approximately 31 saturation points and 23 lightness points. the saturation intervals do not overlap, so the states form two deliberately separated populations rather than positions along a continuous pastel to neon gradient.

these names describe operational states within techno mode; they are not proposed as universal definitions of pastel and neon. hsl is retained here to document the reference implementation, but it is not perceptually uniform. a perceptual implementation should transform generated colors into oklch or cielab before evaluating distances and conflict.

6. the strange parameter

let the interface value strange range from 0 to 100. its normalized form is:

w = strange / 100    0 ≤ w ≤ 1

the parameter w controls the magnitude of permitted chromatic deviation. in the reference implementation it affects both hue spread and state probability. it therefore functions similarly to a mutation amplitude or temperature control: higher values enlarge the available output space without changing the underlying band field.

6.1 hue spread

the total permitted hue spread is:

sₚ(w) = 80° + 220° × w

thus:

sₚ(0) = 80°    sₚ(1) = 300°

this corresponds to 22.2% through 83.3% of the hue circle. if spread is centered on a source hue g, a cell hue may be sampled as h = (g + εₕ) mod 360°, where εₕ is drawn uniformly from [−sₚ(w)/2, sₚ(w)/2]. this equation makes explicit both the meaning of spread and the wraparound behavior at 0° and 360°.

at large values of w, generated hues may more closely resemble another conventional palette family than their source grammar. this is an expected form of mutation, not a change to the source grammar itself.

6.2 pastel probability

let d, e be independent uniform random variables on [0, 1]. the reference condition assigns a cell to the pastel state when d > 0.38 + 0.24we. the resulting marginal probability is:

Pr(pastel | w) = 0.62 − 0.12w

consequently:

Pr(pastel | 0) = 0.62    Pr(pastel | 1) = 0.50

the neon probability is:

Pr(neon | w) = 0.38 + 0.12w

increasing strange therefore moves the binary state distribution from a pastel bias toward an equal split. it does not make the state distribution more random in an absolute sense; instead, it increases its binary shannon entropy toward the one bit maximum. the entropy H(w) is computed from p(w) = 0.62 − 0.12w.

7. two stage generation

techno mode strictly separates structure from chromatic deviation.

stage 1: deterministic structure

  1. construct the grid.
  2. assign each cell to b(r,c) = (r+c) mod n.
  3. assign structural roles, including target and support cells.
  4. select or receive a source harmony grammar.

stage 2: bounded color generation

  1. associate each band or cell role with a source hue.
  2. calculate w and sₚ(w).
  3. sample the pastel or neon state.
  4. sample hue, saturation, and lightness within the permitted distributions.
  5. convert the result to a perceptual color space for evaluation.

randomness never feeds back into stage 1. therefore every output shares a common spatial topology even when its color realization changes substantially.

for reproducibility, every generation should record: {r, c, n, g, w, seed, color space version}. a fixed pseudorandom seed makes the output deterministic and allows exact comparisons between parameter settings.

8. measuring deviation and conflict

bounded input ranges alone do not establish that an output contains controlled visual conflict. to make the claim testable, techno mode distinguishes deviation from the source grammar from perceptual contrast among generated cells.

8.1 circular hue distance

for hue angles a and b, define the shortest circular distance as:

dₕ(a,b) = min(|a − b|, 360° − |a − b|)

8.2 grammar deviation

let gᵢ be the nominal source hue assigned to cell i, and hᵢ its generated hue. mean hue deviation is:

dₛ = (1/m) × ∑ dₕ(hᵢ, gᵢ)    summed over i = 1 to m

where m is the number of evaluated cells.

if rotation of the entire source template is permitted, a rotation invariant measure dₛ* minimizes the mean hue distance over all global rotations θ ∈ [0, 360). the first measure evaluates mutation from the exact initialized palette; the second evaluates deformation of the underlying interval structure independent of global rotation.

8.3 local perceptual conflict

represent the generated color of cell i by a perceptual coordinate xᵢ, preferably in oklab/oklch or cielab. let E be the set of adjacent cell pairs. a basic local contrast score is:

cₛₒₚₕ = (1/|E|) × ∑ δ(xᵢ, x⸉)    for (i,j) ∈ E

where δ is a specified perceptual color distance function.

because high contrast is not automatically equivalent to aesthetic conflict, a more specific model may define a target conflict interval [δ₁, δ₂]. pairwise values below δ₁ are low contrast, values within the interval are controlled conflict, and values above δ₂ are extreme separation. these thresholds should be determined empirically rather than asserted as universal constants.

8.4 multiscale measures

conflict may be summarized at several levels:

  • cell level: deviation of one generated color from its source;
  • neighbor level: perceptual distance across an adjacency edge;
  • band level: within band dispersion and between band separation;
  • grid level: overall color dispersion, state balance, and spatial autocorrelation.

for band b, within band dispersion vᵇ is computed as the mean squared perceptual distance from the band centroid μᵇ over all cells iᵇ in band b. these measures make it possible to test the main claim of the system: as w increases, chromatic deviation and some forms of perceptual conflict should increase, while spatial band membership remains invariant.

9. operational definition of controlled visual conflict

for techno mode, controlled visual conflict is present when all three of the following conditions hold:

  1. structural invariance: band membership and compositional roles are unchanged across generations.
  2. bounded deviation: every sampled color remains within distributions explicitly determined by w, its source hue, and its chromatic state.
  3. measurable tension: the generated palette produces nonzero grammar deviation and a specified distribution of perceptual differences at the cell, neighbor, band, or grid level.

this can be expressed schematically as CVC(p) = αdₛ + βcₛₒₚₕ + γcᵇₕₙₛ + ηH, subject to invariant structural field b. the coefficients are not universal aesthetic constants. they are model parameters that must be selected for a particular implementation or estimated through perceptual studies.

the equation is therefore a measurement framework, not a claim that aesthetic experience can be reduced completely to one scalar.

10. hypotheses and validation

the model supports several falsifiable hypotheses:

  • H1: mean grammar deviation increases monotonically with w.
  • H2: pastel to neon state entropy increases with w, approaching one bit.
  • H3: spatial autocorrelation associated with band membership remains stable across values of w.
  • H4: perceived conflict increases with measured grammar deviation and local perceptual contrast, up to a saturation point.
  • H5: removing the modular band field while holding the color distributions constant reduces perceived structural coherence.

a validation study could compare matched images generated from identical seeds at several values of w. participants could rate perceived coherence, conflict, visual interest, and randomness. an ablation condition could then randomize band membership while preserving the same palette distribution. this would test whether the claimed sense of controlled instability comes from the deterministic substrate, not from the bounded palette alone.

11. conclusion

techno mode is a structured stochastic system for computational color composition. its principal contribution is the separation of an invariant modular spatial field from a mutable chromatic layer. a restricted source grammar establishes an initial palette logic; a bimodal state model separates pastel and neon behavior; and the normalized strange parameter expands hue mutation while moving state selection toward maximum binary entropy.

the system does not propose that visual conflict can be inferred from hue spread alone. instead, it provides a framework in which source grammar deviation, perceptual adjacency contrast, band dispersion, and state entropy can be measured separately. this refinement makes controlled visual conflict an empirically testable proposition:

controlled visual conflict = invariant spatial order + bounded, measurable chromatic deviation

techno mode consequently differs from conventional harmony prescriptions in both purpose and operation. it does not seek a single resolved palette. it defines a reproducible family of compositions in which chromatic instability may increase without dissolving the structure that makes the instability legible.
